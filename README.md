# 🏧 ATM Simulation (Python)

A simple ATM simulation project written in Python. This program demonstrates basic banking operations using a command-line interface.

---

## 📌 Features

* 🔐 User authentication (PIN-based login)
* 💰 Check account balance
* ➕ Deposit money
* ➖ Withdraw money
* 📄 View transaction details (basic)
* 🚪 Exit system safely

---

## 🛠️ Technologies Used

* Python (Core concepts)
* File handling / variables (depending on implementation)
* Conditional statements and loops

---

## ▶️ How to Run

1. Make sure Python is installed
2. Download or clone this repository
3. Open terminal in project folder
4. Run the program:

```bash
python atm.py
```
